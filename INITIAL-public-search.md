## FEATURE:

- Search by username - API - GET https://api.github.com/users/{username}
- Display public profile:
  - Avatar, meno, username
  - Bio, location, website
  - Stats: Public Repos | Followers | Following
  - Join date
  - Repos count
- Responsive design (mobile + desktop) - provided in EXAMPLES
- Light mode / Dark mode
